# Recursive-Problems-Collection
This repository contains a comprehensive collection of recursive function implementations in Python. Each function demonstrates how recursion can be used to solve classical problems in mathematics, strings, arrays, and more. This is ideal for learning recursion, preparing for coding interviews, and practicing problem-solving skills.

## Key Features

Mathematical Recursion: Factorial, Fibonacci, sum of digits, powers, triangular numbers, HCF/GCD.

Array Operations: Find min/max, sum, sum of squares, reverse, subsets, permutations, prime numbers in array.

String Operations: Reverse, check palindrome, remove duplicates, longest common prefix, character count.

Search & Sort: Recursive binary search, merge sorted arrays, check if array is sorted.

Other Classic Problems: Tower of Hanoi, pattern printing, convert decimal to binary, calculate nth power of 2.
